Plugin Documentation

Goals available for this plugin:

Goal Description
snagjar:dependencies No description.
snagjar:deploy No description.
snagjar:help Display help information on snagjar-maven-plugin.
Call mvn snagjar:help -Ddetail=true -Dgoal=<goal-name> to display parameter details.
snagjar:install No description.
snagjar:log No description.

System Requirements

The following specifies the minimum requirements to run this Maven plugin:

Maven 3.3.9
JDK 1.8
Memory No minimum requirement.
Disk Space No minimum requirement.

Usage

You should specify the version in your project's plugin configuration:

<project>
  ...
  <build>
    <!-- To define the plugin version in your parent POM -->
    <pluginManagement>
      <plugins>
        <plugin>
          <groupId>net.adamcin</groupId>
          <artifactId>snagjar-maven-plugin</artifactId>
          <version>0.9.0-SNAPSHOT</version>
        </plugin>
        ...
      </plugins>
    </pluginManagement>
    <!-- To use the plugin goals in your POM or parent POM -->
    <plugins>
      <plugin>
        <groupId>net.adamcin</groupId>
        <artifactId>snagjar-maven-plugin</artifactId>
        <version>0.9.0-SNAPSHOT</version>
      </plugin>
      ...
    </plugins>
  </build>
  ...
</project>

For more information, see "Guide to Configuring Plug-ins"
